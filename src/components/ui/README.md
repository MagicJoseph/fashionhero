# shadcn/ui components

This folder will be populated as you add shadcn components:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
```

The folder structure and configuration is already set up in `components.json` —
just run `add` and the component appears here.

The shadcn skill (loaded automatically via `.claude/skills/shadcn-ui/`) helps
Claude pick the right components and follow shadcn conventions when generating code.
